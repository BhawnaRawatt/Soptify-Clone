🎵 Spotify Clone

A simple Spotify Clone built using JavaScript, HTML, and CSS.
This project replicates the basic UI and music player functionality of Spotify, including play, pause, next, previous, and progress controls.

🚀 Features
🎶 Play and pause songs
⏭️ Next and previous track controls
📀 Music progress bar
🎧 Dynamic song selection
🎨 Responsive and modern UI

🛠️ Technologies Used
HTML5
CSS3
JavaScript
